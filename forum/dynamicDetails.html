WaveLicker : ViceEtVersa (Modular Synth)

Dual Flip-Flop daisy chained if no jack plugged in 2nd entry.

This module trig a gate when output is high since output is high again. It is basically a clock divider (1/2 for each part and 1/4 on second part if no jack is plugged in 2nd entry). it can also be used to octave down an oscillator (one octave for each part and 2 octaves if daisy chained). For each part, it is possible to reset the flip-flop by entering a high signal on RES. The module don't flip-flop since RES is high. RES can be used to "mute" the trigs. www.wavelicker.com
